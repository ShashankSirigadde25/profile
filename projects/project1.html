Python Web Scraping - Music Data Mining

Problem Statement:

How can we automatically extract and analyze trending music data from JioSaavn?

Technologies Used:

Python, BeautifulSoup, Requests, Pandas, CSV

Project Description:

This project scrapes trending songs, artists, and albums from JioSaavn using **BeautifulSoup** and **requests**. The data is structured and saved in a CSV file for analysis.

Key Features:

  • Extracts **Top 50 Trending Songs**
  • Identifies **Popular Artists & Albums**
  • Saves structured data in **CSV format**
  • Uses **regular expressions** to clean extracted data

Challenges Faced:

  • Dynamic content loading by JioSaavn required **handling JavaScript-rendered pages**.
  • Parsing HTML efficiently while **avoiding anti-scraping mechanisms**.

Results & Impact:

This system automates music data collection, saving **80% of manual effort** in research. It provides structured data for **trend analysis and recommendations**.
