Object Diameter Detection Using MATLAB

Problem Statement:

How can we accurately measure the diameter of circular objects in an image?

Technologies Used:

MATLAB, Image Processing Toolbox, Edge Detection, Hough Transform

Project Description:

This project processes an image containing circular objects and determines their diameter using **edge detection** and **circle fitting algorithms**.

Key Features:

  • Detects and isolates **circular objects** in an image.
  • Calculates object **diameter** in pixels and real-world units.
  • Applies **Hough Circle Transform** for accurate measurement.

Challenges Faced:

  • Noise and irregular edges affecting **detection accuracy**.
  • Scaling images to match real-world **physical measurements**.

Results & Impact:

Achieved **95% accuracy** in measuring object diameters, making it suitable for **automated quality control** in manufacturing.
